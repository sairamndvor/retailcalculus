# Retail Calculus Website

## Overview
A modern redesign of the Retail Calculus website - a company that provides cloud-hosted software solutions for traditional furniture and big-ticket retailers.

## Project Structure
- `index.html` - Main HTML file with all page sections
- `style.css` - Complete CSS styling with modern design system
- `script.js` - JavaScript for animations and interactivity

## Features
- Responsive design for all screen sizes
- Modern gradient color scheme
- Smooth scroll navigation
- Animated floating cards in hero section
- Intersection Observer for scroll animations
- Mobile hamburger menu
- Contact form with submission feedback
- Animated stat counters

## Design System
- Primary: #6366f1 (Indigo)
- Secondary: #0ea5e9 (Sky blue)
- Font: Inter (Google Fonts)
- Glassmorphism effects on navbar and cards

## Running the Project
The website is served via Python HTTP server on port 5000.
